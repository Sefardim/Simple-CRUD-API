# Simple-CRUD-API

## Install dependencies

```bash
$ npm i
```

## Running the app

```bash
# development mode
$ npm run start:dev
# production mode
$ npm run start:prod
# development multi mode
$ npm run start:dev-multi 
# production multi mode
$ npm run start:multi 
```

## Application testing

```bash
# run tests
$ npm run test 
```
